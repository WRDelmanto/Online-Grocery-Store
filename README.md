## About
This project aims to develop a **web-based grocery store platform** that allows customers to search for products and place orders online. In today's digital economy, every business needs an online presence, and this platform is designed to enhance the grocery shopping experience for both **customers and administrators**.

### Customer Features:
- Easy browsing by categories
- Real-time inventory visibility
- Streamlined order and payment process

### Admin Features:
- Administrative dashboard for employees
- Product and inventory management
- Order processing and status tracking
- Real-time product availability control

This architecture ensures **scalability**, **responsiveness**, and **maintainability**, making the system suitable for both small and large grocery stores. The platform aims to improve operational efficiency and deliver a seamless experience.

## Technical Stack
Spring-Boot + Vue.js is a **full-stack application** designed with a clear division between frontend and backend, employing modern technologies for high performance and scalability.
- **Frontend**: Vue.js — for reactive UI  
- **Backend**: Spring Boot — for API and business logic  
- **Database**: MySQL (via JPA and Hibernate)  
- **Local Development**: Testable by running the Spring Boot backend and Vue.js frontend simultaneously with proper environment config.

## Usage

### Setting up the Environment (Windows)

**For Spring Boot (Backend)**:

1. Navigate to the Spring Boot directory:
   ```
   cd Spring-Boot+Vue.js/backend
   ```
2. Build and run:
   ```
   mvn spring-boot:run
   ```

**For Vue.js (Frontend)**:

1. Navigate to the Vue directory:
   ```
   cd Spring-Boot+Vue.js/frontend
   ```
2. Install all modules:
   ```
   npm install
   ```
3. Start the application:
   ```
   npm run serve
   ```

Then you can view it at `http://localhost:8080`.
